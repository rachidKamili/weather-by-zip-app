# Weather app (by zip-code)

Search by Zip Code for local weather conditions. Using [Open Weather Map API](https://openweathermap.org/api).

### Architecture
* MVP

### Dependency injection
* Dagger2
* ButterKnife

### HTTP client
* Retrofit2 -> RxJava2 + RxAndroid
* OkHttp

### Converter
* Gson
